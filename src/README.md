# TrailForge — Adventure Gear E-Commerce

## Theme Description
**TrailForge** is a premium, cinematic e-commerce platform dedicated to outdoor adventure gear. The website's aesthetic revolves around a deep, luxury "Forest & Slate" theme, utilizing dark forest-greens, slate grays, and warm golden "saffron" accents to evoke a feeling of high-end outdoor exploration. It features dynamic, immersive masonry gallery layouts, soft shadows, sleek hover animations, and a responsive design that focuses on visually rich product showcases.

## Technologies Used
- **HTML5 & CSS3**: Semantic structure and advanced responsive styling (Flexbox, CSS Grid/Masonry, CSS Variables).
- **Vanilla JavaScript**: Core logic for the shopping cart, dynamic product catalog rendering, theme toggling, and smooth cursor physics.
- **React.js**: For building the interactive, state-driven Testimonials carousel component.
- **LocalStorage API**: For persisting the shopping cart state, user authentication sessions, and dark/light mode preferences across page reloads.

## Additional Features Implemented
- **Premium Custom Cursor**: A custom-built, silky smooth trailing cursor effect with interactive hover detection to enhance the immersive feel.
- **Cinematic Masonry Gallery**: A fully responsive, asymmetric image grid showcasing high-quality outdoor lifestyle photography and gear.
- **Dark/Light Mode Toggle**: Built-in theme switcher that completely alters the site's palette from a deep forest night to a crisp, warm daylight mode.
- **Dynamic Shopping Cart**: Real-time cart updates, price calculation, and an interactive modal interface.
- **Client-side Authentication**: A clean, togglable login/signup UI with session memory.
- **Smooth Scroll Animations**: Reveal-on-scroll effects applied to sections for a polished presentation.
